# SneakHub UI/UX Design

## Overview

SneakHub is a modern high-fidelity mobile e-commerce application designed for purchasing premium sneakers.

## Features

- Home Screen
- Product Details Screen
- Shopping Cart
- Modern UI
- Responsive Mobile Layout
- Easy Navigation

## Tools Used

- Figma
- Iconify
- Unsplash
- Figma AI

## Design System

- Font: Poppins
- Primary Color: Black
- Accent Color: Red
- Background: White

## Screens

- Home
- Product Details
- Cart
